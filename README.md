# Final Project Portfolio

This project is a landing page created as part of the **Final Project** for **MDIA-4003**.

The purpose of this project is to present a simple portfolio landing page built with **Next.js**. Please note that this repository only contains the landing page for the course project.

For the full portfolio, visit:  
[https://www.primcharlin.work/](https://www.primcharlin.work/)

## About This Project

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

The project focuses on creating a clean and responsive landing page that represents my personal brand and design work.

## Tech Stack

- Next.js
- React
- TypeScript
- CSS / Tailwind CSS
- Vercel

## Getting Started

First, install the dependencies:

```bash
npm install

Then, run the development server:

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev

Open http://localhost:3000 with your browser to see the result.

You can start editing the page by modifying:

app/page.tsx

The page auto-updates as you edit the file.

Project Note

This project is submitted for MDIA-4003 Final Project and is intended as a course landing page only.

The complete portfolio website can be viewed here:
https://www.primcharlin.work/
```
